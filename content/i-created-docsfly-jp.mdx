## Docsflyとは？

DocsflyはNext.jsとshadcn/uiで構築されたドキュメントフレームワークです。テンプレートを使用して、瞬時に美しいドキュメントを作成できます。

### 技術スタック

- Next.js
- Tailwind CSS
- shadcn/ui
- TypeScript
- Lucide

## Docsflyの使い方

とても簡単です。次のコマンドを実行して、いくつかの質問に答えるだけで作成できます。

```bash
npx create-docsfly-app # お使いのパッケージマネージャーに置き換えてください
```

その後、docsフォルダーにドキュメントを書くだけです。以上です！

## Docsflyが開発された理由

私は、モダンで機能豊富なドキュメントフレームワークが欲しかったのです。React、Next.js、shadcn/uiで構築されたモダンなドキュメントフレームワークが欲しかったのです。

## まとめ

Docsflyはモダンで機能豊富なドキュメントフレームワークです。ぜひ試してみてください！
